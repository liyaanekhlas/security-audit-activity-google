# security-audit-activity-google
Portfolio project from the Google Cybersecurity Certificate to conduct an internal security audit.

## Overview
This project was completed as part of the Google Cybersecurity Certificate.

In this activity, I reviewed a fictional organization's scope, goals and risk assessment report and then perform an internal audit by completing a controls and compliance checklist to identify risks, evaluate compliance, and recommend security improvements.

## Audit process
To complete the activity, I reviewed the scope, goals, and risk assessment report of Botium Toys(a fictional company) provided in the course. Further I evaluated existing security controls and compliance using a provided checklist template.

Based on the findings, recommendations were made to improve the organization's overall security posture.
